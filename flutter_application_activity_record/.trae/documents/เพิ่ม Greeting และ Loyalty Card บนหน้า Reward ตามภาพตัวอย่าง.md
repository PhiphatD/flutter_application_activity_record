## เป้าหมาย
- เพิ่มส่วนบนหน้า Reward เป็นข้อความทักทาย “Hi ชื่อผู้ใช้” และ “Welcome to Rewards” พร้อมการ์ดบัตรสรุปคะแนนแบบ Loyalty Card ตามภาพตัวอย่าง

## การเปลี่ยนแปลง
- เพิ่มตัวแปร `_userName` สำหรับแสดงชื่อผู้ใช้ (mock)
- เพิ่ม `_buildGreetingRow()` แสดงข้อความทักทาย 2 บรรทัดด้วยฟอนต์ Poppins
- เพิ่ม `_buildLoyaltyCard()` การ์ดคะแนนพื้นหลังไล่เฉดสีน้ำเงิน มีป้ายระดับ “Gold”, ตัวเลขคะแนนจาก `_userPoints` แบบตัวหนา, บรรทัดอธิบาย, และ Expiry
- ปรับลำดับ UI ใน `RewardScreen` ให้เรียง: AppBar → Search → Greeting → Loyalty Card → สวิตช์มุมมอง → Grid รายการ
- ใช้สีและสไตล์สอดคล้องธีม: น้ำเงิน `#4A80FF`, ฟอนต์ Poppins/Kanit, มุมโค้ง, ไม่มีการซ้อนองค์ประกอบอินเทอร์แอคทีฟ

## ทดสอบ
- เห็นทักทายผู้ใช้และการ์ดคะแนนด้านบน
- คะแนนแสดงตาม `_userPoints` พร้อมรูปถ้วยทางขวาและ Expiry
- ส่วน Reward items ยังคงเป็น Grid 2 คอลัมน์ และ Redeem ทำงานตามเดิม
