## เป้าหมาย
- ปรับการ์ด Reward items ให้เหมือนภาพตัวอย่าง: รูปสินค้าที่ด้านบน, ชื่อด้านล่าง, และปุ่ม `Use` แบบ outlined พร้อมไอคอนเหรียญและจำนวนคะแนน
- ยังคง Grid 2 คอลัมน์ต่อแถว

## การเปลี่ยนแปลง
- รีแฟกเตอร์ `_RewardItemCard` ให้เป็นเลย์เอาต์แนวตั้ง (Column):
  - `ClipRRect` ภาพเต็มความกว้างการ์ด, สูงคงที่, `fit: cover`
  - ข้อความชื่อรางวัล 1–2 บรรทัด ellipsis
  - ปุ่ม `Use` แบบ `OutlinedButton` เต็มความกว้าง มีไอคอนเหรียญและจำนวนคะแนน
- ปรับ `GridView` ให้ `childAspectRatio` เหมาะกับเลย์เอาต์ใหม่

## ทดสอบ
- Grid แสดง 2 x 2 ต่อแถว, การ์ดแต่ละใบมีรูปด้านบนและปุ่ม Use ตามดีไซน์
- ปุ่ม `Use` ใช้ได้เมื่อพอยท์พอและมีสต็อก; ทำงานลดสต็อก/คะแนนและย้ายไป Redeemed ตามเดิม
