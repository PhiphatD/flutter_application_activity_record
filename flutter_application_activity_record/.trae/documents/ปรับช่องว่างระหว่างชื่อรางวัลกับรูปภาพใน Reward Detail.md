## จุดที่ต้องปรับ

* ปัจจุบันช่องว่างเกิดจาก `padding` ด้านบนของคอนเทนต์ในหน้า Reward Detail ที่ตั้งไว้ `EdgeInsets.all(20)` ที่ `lib/screens/employee_screens/reward_detail_screen.dart:156`.

## แผนการแก้ไข

1. ลดระยะห่างด้านบนของคอนเทนต์

* แก้บรรทัด `padding: const EdgeInsets.all(20)` เป็น `padding: const EdgeInsets.fromLTRB(20, 12, 20, 20)` เพื่อให้ระยะระหว่างรูปกับชื่อรางวัลกระชับขึ้นอย่างพอดี

1. ปรับระยะภายในกลุ่มชื่อและชิปคะแนนให้สมดุล

* เพิ่ม/ลด `SizedBox(height: 8)` ระหว่างชื่อรางวัลกับแถวหมวดหมู่+stock เพื่อให้เนื้อหาอ่านสบายตา โดยคงโครงเดิมไว้

## โค้ดที่จะเปลี่ยน

* ที่ `reward_detail_screen.dart:156`

```
padding: const EdgeInsets.fromLTRB(20, 12, 20, 20),
```

* ที่บล็อคชื่อรางวัล cเพิ่มตัวเว้นบรรทัดเล็กน้อย (ถ้าต้องการ)

```
const SizedBox(height: 8),
```

## ผลลัพธ์ที่คาดหวัง

* ช่องว่างระหว่างรูปภาพและข้อความ "Starbucks eVoucher 100 THB" ลดลง ดูกระชับและสมดุลกับดีไซน์โดยรวม
